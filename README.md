🌍 Travel Companion App

A simple and user-friendly Android application developed using Kotlin and XML that helps international travellers perform essential conversions quickly and efficiently.

📱 Features

The app supports multiple conversion categories:

💱 Currency Conversion
USD ↔ AUD
USD ↔ EUR
USD ↔ JPY
USD ↔ GBP

⛽ Fuel Efficiency
MPG ↔ KM/L

🧪 Volume Conversion
Gallon ↔ Liter

📏 Distance Conversion
Kilometer ↔ Nautical Mile

🌡️ Temperature Conversion
Celsius ↔ Fahrenheit
Celsius ↔ Kelvin
Fahrenheit ↔ Kelvin

🎨 User Interface

Clean and simple layout using XML
Dropdown (Spinner) menus for selecting units
Input field for user values
Convert button to perform calculations
Result display area
Color-enhanced UI for better user experience

🛠️ Technologies Used

Kotlin – Application logic
XML – UI design
Android Studio – Development environment
Android SDK – App framework

⚙️ How It Works

Select a category (Currency, Fuel, Volume, Distance, Temperature)
Choose From and To units
Enter the value
Click Convert
View the result instantly

🛡️ Validation & Error Handling

The app includes robust validation to ensure reliability:

Prevents empty input
Prevents non-numeric values
Handles identity conversions (e.g., USD → USD)
Restricts negative values for fuel, volume, and distance
Displays appropriate error messages and Toast notifications

🧪 Example Conversions

Input	Conversion	Output
10 USD → AUD	Currency	15.50 AUD

5 Gallon → Liter	Volume	18.93 L
